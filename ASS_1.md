# Review Questions

## 1. What is the Internet?
The Internet is a worldwide network that connects computers and devices, allowing people to communicate and share information.

## 2. Explain the difference between the Internet and the World Wide Web.
The **Internet** is the global network that connects devices. The **World Wide Web (WWW)** is a service that uses the Internet to access websites and web pages through web browsers.

## 3. What is the role of an ISP?
An **Internet Service Provider (ISP)** is a company that provides individuals and organizations with access to the Internet. Examples include MTN, Airtel, Glo, and 9mobile.

## 4. What is an IP address?
An **IP address (Internet Protocol address)** is a unique numerical address assigned to a device connected to the Internet. It helps identify and locate that device on the network.

## 5. What does DNS do?
**DNS (Domain Name System)** translates human-readable website names into IP addresses. For example, it converts `www.google.com` into the numerical address computers use to find the website.

## 6. Explain the difference between HTTP and HTTPS.
- **HTTP (HyperText Transfer Protocol):** Transfers data between a web browser and a website but does not provide encryption.
- **HTTPS (HyperText Transfer Protocol Secure):** Encrypts the data being transferred, making communication more secure and protecting users' information.

## 7. Mention three benefits of the Internet.
1. It enables easy communication through emails, messaging apps, and video calls.
2. It provides access to information and educational resources.
3. It allows users to enjoy online services such as shopping, banking, and entertainment.

## 8. List three Internet security best practices.
1. Use strong and unique passwords.
2. Avoid clicking on suspicious links or downloading unknown files.
3. Keep your devices and software updated with the latest security patches.
